NSW RENTAL PRESSURE & HOUSING INEQUALITY PROJECT
PROJECT STRUCTURE & CHRONOLOGY

==================================================
PROJECT FOLDER STRUCTURE
==================================================

data/
- raw/
  Original datasets received from the team

- cleaned/
  Intermediate cleaned datasets generated during preprocessing

- final_tableau/
  Final Tableau-ready datasets


scripts/
- validation/
  Scripts used to inspect raw files, identify header rows, validate structure and preview datasets

- cleaning/
  Scripts used for:
  - null handling
  - duplicate removal
  - type conversion
  - standardisation
  - NSW filtering
  - feature cleanup

- merging/
  Scripts used to:
  - aggregate metrics
  - merge datasets
  - generate Tableau-ready outputs


docs/
- Reserved for additional project documentation if needed


handover/
- Final datasets
- Data dictionary
- Project summary
- Structure and chronology documentation


requirements.txt
- Python package dependency list generated using pip freeze
