# Dashboard — The Renter's Tax NSW

Live Tableau dashboard for the persuasion pitch. Built on the EDA findings, designed as a decision-support tool — not just a reporting view.

## Open the dashboard
🔗 **[The Renter's Tax NSW — Tableau Public](https://public.tableau.com/app/profile/md.abtab.karim.kabbo/viz/TheRentersTaxNSW/Dashboard1?publish=yes)**

## What it shows
- **Bond loss by SEIFA decile** — the headline gradient (37% → 13%)
- **Geography of crisis map** — bond loss rate across NSW postcodes
- **The 8 squeezed suburbs** — disadvantaged postcodes paying above-median rent
- **Affordability slider** — interactive view of what essential workers can afford

## How it's used in the pitch
The dashboard powers the live demo in slide 7. The presenter drags the SEIFA filter and the budget slider while the audience watches the bond-loss gradient and the affordability map respond in real time.

## Data sources
- NSW Fair Trading rental bond data (2025)
- ABS SEIFA 2021 — postal area indexes
- 636 NSW postcodes · 303,111 lodgements · 310,428 refunds
